Linux Text Editor Console and Graphical

Bismillahirrahmanirrahim.

Console

Text editors that can be used in a command line interface environment.

  1. nano - small, friendly text editor inspired by Pico

  2. vim - A full-featured editor with syntax highlighting

  3. emacs - the extensible self-documenting text editor( high learning curve)

  4. mc - Midnight Commander provides an internal text editor (mc -e,mcedit)

  5. jed - editor for programmers (textmode version)

  6. jupp/joe/joe-jupp - user friendly full screen text editor with keybindings/modes, hex editor, snytax highlighting, search/replace, regex, macros and status lines.

Graphical

Text editors that can be used in a graphical environment.

  1. gedit - Default text editor of Gnome desktop environment. Aiming at simplicity by default, can be configured as full fledged integrated development environment through various plugins provided by gedit-plugins package.

  2. geany - Advanced text editor with basic features of integrated development environment and has only few dependencies on other packages.

  3. pyroom - Full screen text editor for distraction free writing, with configurable colour-schemes and keybindings.

  4. scite - A GTK+ based programmers editor, uses Scintilla editing component.

  5. kwrite - Default text editor of KDE Software compilation, provides syntax highlighting and ability to export documents to PDF, HTML, PostScript among other features.

  6. kate - Acronym for KDE Advanced Text Editor, can be turned to full featured integrated development environment, providing extendable (via XML) syntax highlighting, session management and other features.

  7. mousepad - Default editor of Xfce desktop environment, intended to be, simple, fast and easy to use.

  8. leafpad - A minimalist GTK+ based text editor, for basic text editing needs.

  9. GUI version of vim, provided by packages vim-gtk3, vim-gtk, vim-athena.

  10. Emacs - The default Emacs package includes an interface for X11.

  11. jedit - very powerful/flexible (like emacs) and also easy to use (like notepad). Has many plugins for specialized needs.

I hope this will help you. Barakallahu fiikum.
