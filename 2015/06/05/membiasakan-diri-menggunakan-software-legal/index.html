Membiasakan Diri Menggunakan Software Legal Freeware / Open Source

Bismillahirrahmanirrahim.

Artikel ini saya tunjukan untuk setiap orang yang telah menyadari pentingnya penggunaan software legal. Dan artikel ini pun ditunjukan sebagai ajakan bagi pengguna Windows (dan Mac OS X) untuk menggunakan software legal. Apabila Anda telah memiliki kesadaran mengenai pentingnya penggunaan software legal, maka jagalah. Ini adalah sikap yang baik dan membawa kemaslahatan, akhlak baik yang sangat langka pada zaman ini.

Sesuatu yang Baik harus dimulai dari yang Baik pula. Apalagi jika software-software tersebut digunakan untuk mencari nafkah dan rezeki.

Agar dapat membedakan dan memahami lebih dalam saya coba bantu jelaskan apa itu definisi Software, Freeware, dan Open Source.

Software

adalah istilah untuk menyebut sekumpulan data atau intruksi komputer yang membantu manusia dalam tujuan tertentu. Mudahnya begini, software bukan istilah untuk menyebut segala perangkat fisik yang dapat disentuh, jadi software itu bukan hardware atau benda yang dapat disentuh dengan anggota tubuh atau benda lainnya. Software adalah istilah untuk menyebut benda-benda nonfisik seperti antivirus, video game, download manager, web browser, office suite, email client, dan sebagainya. Software biasa disebut dengan nama lain aplikasi, utility, tool, program, dan sebagainya.

Freeware

adalah software yang gratis, berlisensi bebas dipakai, tetapi tidak bebas dimodifikasi, dan tidak menyertakan kode sumber ketika dipublikasikan. Contoh freeware adalah AIMP, Skype, ImgBurn, Recuva, Foxit Reader dan lain-lain.

Open Source

adalah software yang berlisensi open source, yakni bebas dipakai, bebas dimodifikasi, bebas disebarluaskan, dan menyertakan kode sumber ketika dipublikasikan. Contoh open source software adalah Firefox, GIMP, Inkscape, VLC, dan Libreoffice.

Catatan:
Freeware itu bukan open source software, dan open source software bukan freeware. Namun keduanya termasuk software legal.

……….

Kembali pada persoalan diatas. Jika kita sadar bahwa penggunaan software ilegal itu tidak boleh dan tidak baik, maka berusahalah dengan sungguh dan sekuat tenaga untuk mencari software legal sebagai penggantinya. Tidak harus membeli, software legal bisa diperoleh dalam bentuk freeware maupun open source software.

Beberapa poin yang harus kita ambil:

  1. Milikilah keinginan yang kuat untuk mencari pengganti software komersial yang belum mampu Anda beli.

  2. Berusaha meninggalkan dan tidak mencari crack, patch, atau serial number ilegal ketika Anda membutuhkannya.

  3. Milikilah rasa percaya diri yang tinggi menggunakan software legal, bahwa secara umum seluruh software berbayar sudah memiliki padanannya dari kalangan freeware maupun open source software.

  4. Bagi pengguna Windows / Mac OS X yang masih memakai bajakan, segeralah Anda menggantinya dengan yang asli legal atau beralih menggunakan Linux.

  5. Untuk pengguna Windows, ketahuilah tempat memperoleh software gratis yang legal seperti http://softpedia.com, http://download.cnet.com, http://en.softonic.com/, http://www.pcadvisor.co.uk/downloads/windows/, http://filehippo.com/, http://www.tucows.com/downloads, http://sourceforge.net, atau http://freshmeat.net.

  6. Bagi pengguna Windows, ketahuilah situs-situs open source software untuk Windows semisal http://opensourcewindows.org/.

  7. Bagi pengguna Mac OS X, ketahuilah situs-situs open source software untuk Mac OS X semisal http://opensourcemac.org/.

  8. Untuk pengguna Linux, ketahuilah situs-situs perbandingan aplikasi Windows – Linux sebagai penerapan sikap pertama di atas. Misalnya http://alternativeto.net, http://osalt.com, http://www.linuxalt.com/, http://www.linuxsoftware.org, http://www.linuxrsp.ru/win-lin-soft/table-eng.html, http://www.linuxlinks.com/article/20070701111340544/Equivalents.html, http://wiki.linuxquestions.org/wiki/Linux_software_equivalent_to_Windows_software, atau https://help.ubuntu.com/community/ListOfOpenSourcePrograms.

  9. Bagi pengguna Linux, ketahuilah situs-situs tempat Anda bisa memperoleh paket program secara offline semisal http://apt-web.tk (Ubuntu), http://pkgs.org (semua distro), http://rpm.pbone.net/ (Fedora, openSUSE, Mageia), http://rpmfind.net (Fedora, openSUSE, Mageia), dan lain-lain.

  10. Milikilah keinginan untuk membeli software asli, jika memang Anda membutuhkannya.

Jika Anda telah membiasakan diri menggunakan software legal dengan pemahaman konsep dan sikap yang bagus, maka mudah bagi Anda untuk menggunakan Linux. Ingat! kita hanya perlu membiasakan, kaku itu karena tidak terbiasa. Kunci kekuatan orang-orang yang betah dan kukuh menggunakan Linux selama bertahun-tahun adalah pembiasaan pada diri mereka yang di atas rata-rata untuk menggunakan software legal. Pembiasaan yang sesungguhnya itu berada di dalam hati kecil, biasa mengakui keadilan dan selalu berusaha berbuat baik. Silakan kita memulai dari diri kita masing-masing. Barakallahu fiikum.
