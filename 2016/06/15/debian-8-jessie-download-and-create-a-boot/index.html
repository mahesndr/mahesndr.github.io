Debian 8 Jessie - Download and Create a bootable USB

Bismillahirrahmanirrahim.

Download Debian 8 Jessie

Download Debian GNU/Linux 8 Jessie ( released on 25 April, 2015 )

Create a bootable USB

Create a bootable USB for installing Debian. For example, if you create it with Fedora standard function, refer to follows. You can adjust the system you are using.

  1. Install Unetbootin

  2. Insert an empty USB Flash Drive in USB Port.

  3. Next. Run Unetbootin, choose Diskimage-ISO-Browse to the ISO Image, see the pictures.

  4. Reboot Now or Exit

  5. Set your system BIOS or boot menu to boot from the USB device and enjoy your favorite Live Linux on USB.
