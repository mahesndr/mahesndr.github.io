SSH Server : Password Authentication

Configure SSH Server to manage a server from the remote computer. SSH uses 22/TCP.

Configuration files

The main configuration files are in the directory /etc/ssh :

  • ssh_config : client configuration file

  • sshd_config : server configuration file

Install and configure sshd with Password Authentication.
