Debian 8 Jessie Install

Bismillahirrahmanirrahim.

  1. Set your system BIOS or boot menu to boot from the USB device and enjoy your favorite Live Linux on USB. Next, select installation type on the screen below. ON here, choose Graphical.

  2. Select your language.

  3. Select your location fot timezone.

  4. Select the locale.

  5. Select your keyboard type.

  6. Set your server’s hostname you’d like to set.

  7. Set your network domain name.

  8. Set root password.

  9. Set a user account.

  10. Input a user name used on the system.

  11. Set password for a user in pre-section.

  12. Select a city in your time zone.

  13. This is the section for configure partition. This example select “Guided with VLM” like follows.

  14. Select a disk to create partition.

  15. Select a type of partition layout you’d like to set.

  16. Confirmatin remove existing logical volume data. If it’s OK, answer “Yes”.

  17. It’s the confirmation for configuring partition. If it’s OK, answer “Yes”.

  18. It’s completed to configure partitions. If it’s OK, answer “Yes”.

  19. It’s final confirmation for changing disks. If it’s OK, answer “Yes”.

  20. If you’d like to install more packages in another CD/DVD, answer “Yes”. And the base system will be run.

  21. If you’d like to install more packages in another CD/DVD, If it’s OK, answer “Yes”. If it’s not, answer “No”. I choose “No”.

  22. If you’d like to use a network mirror, answer “Yes”. If it’s not, answer “No”, you can skip in this step and add it after the operating system is installed. I choose “Yes”.

  23. If you selected ‘Yes’ in pre-section, choose a location of mirror site.

  24. Select a mirror site.

  25. Set Proxy if you use it. If not, Continue with empty.

  26. Answer ‘Yes’ to participate package usage survey as an anonymous.

  27. Select software groups to install. This example proceeds with only “Standard system utilties” like follows.

  28. Answer “Yes” if you’d like to install GRUB on MBR.

  29. Select a disk you’d like to install GRUB.

  30. Installation completed. Remove the disk and Continue. Then system will reboot.

  31. After rebooting system, login prompt is shown like follows. Login with a user set in installation.

Barakallahu fiikum.
