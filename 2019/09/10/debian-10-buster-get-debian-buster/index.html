Debian 10 Buster - Get Debian Buster

Bismillahirrahmanirrahim.

Download Debian 10 Buster

Download Debian GNU/Linux 10 Buster ( released on 06 July 2019 ) from the following site.

Create an installation CD/DVD

After downloading ISO of Debian, Burn it to a DVD with an burning application to create an installation disk. If you create it with the function of Windows 7/8/10, refer to follows.

  1. Insert an empty DVD media in DVD drive. Next, right-click the ISO image of Debian and select [Burn Disk image] in the menu.

  2. Click [Burn] button and start burning.

  3. After finishing burning, eject the disk and proceed to the installation step.

I hope this will help you. Barakallahu fiikum.
